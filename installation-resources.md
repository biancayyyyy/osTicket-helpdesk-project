# 📦 Installation Resources ⚙️

This document contains all tools and files used to deploy and configure the osTicket Help Desk system in a Windows Azure Virtual Machine environment.

---

## 🧰 Core Installation Files

- 🎫 **[osTicket Installation Package](https://drive.google.com/uc?export=download&id=1b3RBkXTLNGXbibeMuAynkfzdBC1NnqaD)**  
  Used to install and deploy the osTicket application.

- 🧩 **[Microsoft Visual C++ Redistributable (VC_redist.x86)](https://drive.google.com/file/d/1s1OsGF3-ioO0_9LYizPRiVuIkb3lFJgH/view)**  
  Required dependency for running PHP and osTicket.

- 🗄️ **[MySQL Server 5.5.62](https://drive.google.com/file/d/1_OWh9p7VQLcrB0q_V7qT8yHl0xo5gv7z/view)**  
  Database system used to store ticketing data.

- 🛠️ **[HeidiSQL](https://docs.google.com/document/d/1WovrX2DaS9xkfaSr4LXyB4YnnWpXIgPCMMbbfgHmGVw/edit)**  
  Database management tool used to configure MySQL and manage osTicket database.

---

## 🌐 Local Testing Environment

- 🧑‍💻 **[User Portal (osTicket)](http://localhost/osTicket/)**  
  End-user interface for submitting support tickets.

- 🔐 **[Admin Panel (osTicket SCP)](http://localhost/osTicket/scp/login.php)**  
  Admin dashboard used to manage, respond, and resolve tickets.

---

## ⚠️ Lab Environment Note

This project was completed in a controlled virtual lab environment using Azure Virtual Machines for educational purposes only.
